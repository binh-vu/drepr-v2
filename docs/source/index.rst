|drepr| documentation
==================================

|drepr| is a language for specifying heterogeneous datasets and a library for converting the datasets to RDF or JSON. 

Checkout the :doc:`Getting Started guide <./getting-started>` to learn how to use it.

For feedback and bugs, please open an issue on its `Github repository <https://github.com/binh-vu/drepr-v2>`_.
